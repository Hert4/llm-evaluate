# LLM Evaluation – Sample Size per Metric

## Dịch thuật
- **BLEU:** 200–500 câu  
- **COMET:** 200–500 câu  

## Tóm tắt văn bản
- **ROUGE:** 100–300 bài tóm tắt  
- **G-Eval:** 50–100 mẫu  

## Hỏi đáp (QA)
- **Exact Match (EM) & Token F1:** 300–500 cặp Q&A  

## Gợi ý / Đề xuất
- **NDCG@K:** 100–200 queries  
- **Recall@K:** 100–200 queries  

## Gọi hàm / Tool
- **AST Accuracy:** 100–300 test case  
- **Task Success Rate:** 50–100 task đa bước  

## Lập trình
- **Pass@K:** 50–100 bài toán lập trình  
- **% Issues Resolved (SWE-Bench):** 30–50 issues thực tế  

## Suy luận & Kiến thức
- **Accuracy (Trắc nghiệm):** 200–400 câu hỏi  

## Đa phương thức
- **VQA Accuracy:** 200–400 ảnh / biểu đồ  

## RAG Pipeline
- **Faithfulness:** 100–200 mẫu Q&A  
- **Context Precision & Recall:** 100–200 mẫu  

## An toàn / Safety
- **FactScore:** 50–100 câu trả lời  
- **Attack Success Rate:** 50–100 prompt tấn công  
- **IFEval:** 100–200 instruction  

## Chatbot / Chat
- **ELO Rating:** 300–500 cặp so sánh  
- **Win Rate (AlpacaEval 2.0):** 200–400 câu hỏi
